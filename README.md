## About this project

- Stock trading assistant that uses data from social media websites to give buy/sell recommendations for given stocks.
- Frontend built in React with Tailwind CSS, backend built with Go.
- Uses web-scraping to gather real-time data from X/Twitter.
- Completely free!

## Instructions

1. To run this app locally, first clone the repository to your local machine.
2. In the root directory, run the command `npm install` to gather all needed dependencies.
3. Then, run the app in development mode by running the command `npm start`.
4. Open [http://localhost:3000](http://localhost:3000)!
